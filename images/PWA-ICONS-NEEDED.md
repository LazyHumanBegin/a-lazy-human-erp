# PWA Icons Needed

Please create the following icon files for the PWA to work properly:

## Required Icons (PNG format):

| Filename | Size | Purpose |
|----------|------|---------|
| icon-72.png | 72x72 px | Android small |
| icon-96.png | 96x96 px | Android medium |
| icon-128.png | 128x128 px | Chrome Web Store |
| icon-144.png | 144x144 px | Android medium |
| icon-152.png | 152x152 px | iOS iPad |
| icon-192.png | 192x192 px | Android large, PWA |
| icon-384.png | 384x384 px | PWA splash |
| icon-512.png | 512x512 px | PWA splash, Play Store |

## Optional Icons:
| Filename | Size | Purpose |
|----------|------|---------|
| icon-pos.png | 96x96 px | Shortcut icon |
| icon-transaction.png | 96x96 px | Shortcut icon |
| screenshot-dashboard.png | 1280x720 px | App store listing |

## Design Guidelines:
- Use the EZCubic cube logo (blue #2563eb)
- Square format with rounded corners (maskable)
- Dark or white background
- Simple, recognizable design

## Quick Create (Online Tools):
1. Go to https://www.favicon-generator.org/
2. Or use https://realfavicongenerator.net/
3. Upload a 512x512 image, it generates all sizes

## For now:
The PWA will still work without icons, but the install prompt will show a generic icon.
